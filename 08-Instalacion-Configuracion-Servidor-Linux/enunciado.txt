instalar y configurar un servidor con Ubuntu Server que proporcione servicios básicos de red y desarrollo web.
El proyecto busca afianzar competencias en administración de sistemas, seguridad básica y despliegue de servicios.

Requerimientos mínimos
Sistema operativo: Ubuntu Server (en máquina virtual o real).

Acceso remoto: Configuración de conexión vía SSH.

Servidor web: Instalación y puesta en marcha de Apache2.

Base de datos: Instalación y configuración de MySQL.

Creación de al menos un usuario con contraseña segura.

Acceso desde cliente remoto (Workbench o similar).

Transferencia de archivos: Instalación y configuración de VSFTPD.

Permitir acceso con un usuario propio para la carpeta web.

Soporte de aplicaciones: Instalación de PHP y prueba con un script de información del sistema.

Cliente de conexión: Comprobar acceso remoto desde FileZilla, Workbench y VSCode (SSH).

Entregables
Un informe breve (README.md) con:

 

Comandos utilizados.

Problemas encontrados y soluciones aplicadas.

Descripción de cada servicio instalado.

Sugerencias
Mantén siempre actualizado el sistema con sudo apt update && sudo apt upgrade.

Realiza copias de seguridad de los archivos de configuración antes de modificarlos.

Documenta cada cambio que hagas en tu informe.
